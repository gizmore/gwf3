# CGX#: Regular expressions

In this video we talk about Regular Expressions.


 - delimiter `/`
 - modifiers `/../gumiD`
 - start and end `/^$/iD`
 - character classes `/[^e]/iD`  \w \s \d
 - quantifier `/[abc]?*+[abc]{1,2}/iD`
 - capturing `([abc]{1,3})`
 - non capturing group `(?:str1|str2)`
